# Botsolver 🤖

**Botsolver** is a simple CLI built in **Python** that lets you talk with an AI assistant named Botsolver.
Built as a fun way to learn Python and work with AI APIs from the terminal.

## Requirements

- Python 3.7+
- pip

## Setup & Run

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the app:
   ```
   python botsolver.py
   ```

## Usage

- Type any message and press Enter to chat with Botsolver
- Type `exit` to quit
